# Coffee Table Book — content format

[Coffee Table Book folder — Drive](https://drive.google.com/drive/folders/1ri9JSluD1-Z-ifMzZ-hRXDvk47g7VqTq)
[Coffee Table Books (newer collection)](https://drive.google.com/drive/folders/1Mj89-mdxhFRjEhUTC7CBcU-reEX_JJzZ)

## Why this matters for Mantra4Change

The SoW (May–September 2026) calls out "long-form outputs (e.g. coffee table book support)" as a Phase 1 deliverable. This route is the canonical reference for how Shikshagraha tells micro-improvement leader stories in long-form print.

## Templates available

- **Coffee Table Book Template_V3** ([doc](https://docs.google.com/document/d/1Zlw0S_P_49iaovUkOUQNIi5GbYKqGi9XsykLq6EoK_A/edit)) — most recent (Jan 2025). Canonical format. Fields: Title / Subtitle / Leader Name / Designation / School / Location / Mentored by / The Challenge / What Changed (70-word impact) / Journey of the Youth Leader (100 words) / Testimonial / Project Steps. **Mantra4Change content for the next coffee table book should follow this template exactly.**
- **CoffeeTableBook_CSOs** ([doc](https://docs.google.com/document/d/1A1KUOmUMSgpjTZ_oMn0IjOpmloTb0o-b4zAkJnnV4vU/edit)) — Jun 2024 stub.
- **Coffee Table Book Template** ([doc](https://docs.google.com/document/d/1hIQGQNJlI1zSHmYv7ZO6tozx9QfpRaR4g-UmCC02Ao4/edit)) — Mar 2024 (earlier version).
- **VarthaLeap Members folder** ([folder](https://drive.google.com/drive/folders/1Z6qveDquyxF8WP1vInfZzum9woAFOQpP)) — where CSOs upload their per-leader stories per the template.

## How the brain uses this

When Pratik asks "draft me a coffee table book entry for [leader name]," the brain pulls the V3 template structure, fills it from `wiki/synthesis/` and `wiki/concepts/` content, and produces a draft conforming to the format. Output is written to `output/` (or chat reply if `output_channel: none`).
