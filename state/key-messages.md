# Key messages

**Every piece of Shikshagraha communication should echo at least one of these.** If a draft doesn't, it's off-message — revise before sending.

## The five

1. **Every child deserves a great school** — not just children who can afford one.
2. **India's public school system has the scale. Shikshagraha builds the will.**
3. **School transformation happens when communities act** — not just when governments fund.
4. **Every child deserves an enriching learning experience that prepares them for the future.**
5. **Education must be a collective responsibility and a public priority** — a shared agenda for all of society.

## How to apply

- Treat as a checklist: read your draft, name which of the five it echoes
- Don't try to hit all five in one piece — one well-landed message beats five whispered ones
- The phrasing can shift; the stance can't
- For format-by-format guidance on how messages land in each medium, see [[formats]]

## Why these five and not others

Per [[mission]] § norms-shifting: these five are the public-facing distillation of the seven norm shifts. Each message corresponds to one or more of the existing-norm → new-norm pairs.

If a draft echoes a sixth implicit message that isn't here, that's signal for the weekly review — flag in `learnings/`.
